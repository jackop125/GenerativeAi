Creation Process

Image Generation :

Each item is generated by an algorithm.
The first step is to pick the primary colors for the artwork. User can pick a random HSV value in a range, then a secondary color based off of that. The art itself is then generated by drawing a series of connected lines, with variable thickness. The color and thickness changes between each point.These colors, points, and thickness are then serialized into a code which is then used to render the image. In this way, the meta-data also contains a redundant back-up of the image itself.

Password Generation :

Password Generator will provide randomly generated passwords which would be derived from the images created in the first step. Further the hash of this image will be generated using MD5, SHA256 etc. The generated hash will be further randomized using ROT-13, etc.
